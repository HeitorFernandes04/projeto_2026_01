# 🛠️ Stack Tecnológica

## Backend (SaaS Core)
- **Framework:** Django 5.2.12
- **API:** Django Rest Framework (DRF) 3.17.1
- **Banco de Dados:** SQLite (Dev) / PostgreSQL (Prod Ready)
- **AI/RAG:** ChromaDB 1.5.5 + MCP (Model Context Protocol) 1.27.0
- **Segurança:** SimpleJWT (Autenticação Unificada)

## Frontend Web (Portal Administrativo & B2C)
- **Framework:** Angular 21.2.0
- **Estilos:** SCSS com Variáveis CSS
- **Estado:** RxJS (Programação Reativa)
- **Gráficos:** Chart.js

## Frontend Mobile (Operacional de Pista)
- **Framework:** Ionic 8.5.0 + React 19.0.0
- **Bridge:** Capacitor 8.3.0
- **Ícones:** Lucide React & Ionicons

## Qualidade e DevOps
- **Testes Backend:** Pytest + Pytest-Django
- **Testes Web/Mobile:** Vitest + Cypress
- **Orquestração:** Makefile
